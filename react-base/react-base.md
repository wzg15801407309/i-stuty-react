## 3、React 组件状态

- 函数组件叫做**无状态组件**，类组件叫做**有状态组件**
- 状态（state）即数据
- 函数组件没有自己的状态，只负责数据展示（静）
- 类组件有自己的状态，负责更新 UI，让页面“动”动起来（用户可以看见的变化改变）
